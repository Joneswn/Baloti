Disclaimer

Please note that the vote you are casting on the baloti website is not official..

On the website www.baloti.ch various user data are saved

In a cookie we store the language preference of the user only. When the website is accessed login data are saved in a so-called log file on the server. This mainly concerns: IP address, date and time of access, name of accessed page, web-browser used, operating system used etc.

All these data are protected against unauthorised persons and are only evaluated in the form of anonymous access statistics.

E-mail addresses are used only for direct communication with the user and will under no circumstances be passed on to a third party.

The data about the vote and the user (status, age, gender, canton, etc.) are used for analyses and statistics only in anonymised and aggregated form.
